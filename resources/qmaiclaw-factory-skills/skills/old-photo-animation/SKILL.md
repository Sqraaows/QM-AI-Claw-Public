# 老照片动起来

## 概述

本技能将老照片处理与数字人口播结合，自动完成从修复到生成动态口播视频的全流程：
1. 先调用老照片修复技能对输入老照片进行AI高清修复
2. 基于修复后的图片，自动选择匹配老照片内容的文案
3. 使用真人直接口播方案生成动态视频，自动添加AI配音和字幕

整个流程无需人工干预即可完成，最终输出可直接使用的动态口播视频。

## 工作流程

### 1. 输入检查
- 确认输入包含需要处理的老照片文件
- 如果缺少输入，请求用户补充

### 2. AI高清修复
- 调用老照片修复技能，使用用户指定（或默认流动）轨迹修复老照片
- 获取修复后的高清图片

### 3. 自动匹配文案
- 分析老照片内容（场景、人物、年代等）
- 自动选择一段与内容匹配的文案，可由用户确认或修改
- 如果用户提供了自定义文案，使用用户文案

### 4. 生成口播视频
- 调用数字人口播技能，使用真人直接口播方案
- 基于用户选择（或默认流动）轨迹生成口播视频
- 自动生成AI配音，添加字幕到视频中

### 5. 输出结果
- 返回最终生成的动态口播视频文件
- 同时提供修复后的高清老照片作为副产品

## 平台对接说明

本技能依赖以下两个技能完成处理：
- [old-photo-restoration](../old-photo-restoration/SKILL.md): 老照片AI修复
- [digital-human-voiceover](../digital-human-voiceover/SKILL.md): 数字人口播生成

处理轨迹选择继承自用户输入，默认使用流动轨迹。
