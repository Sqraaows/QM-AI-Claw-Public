# Finance LLM Skill - 金融大模型助手

## 功能描述
专门为金融领域设计的大模型应用技能，涵盖：
- 金融数据分析与解读
- 投资策略建议
- 风险评估与管理
- 财务报表分析
- 市场趋势预测
- 金融术语解释
- 投资组合优化

## 使用场景
- 个人理财规划
- 股票/基金投资分析
- 企业财务分析
- 宏观经济解读
- 金融新闻摘要
- 投资教育内容创作

## 内容创作模板

### 1. 金融知识科普类
**标题公式**: "[数字]+个[金融概念]真相，90%的人都不知道"
**结构**: 痛点引入 → 概念解释 → 实际案例 → 行动建议

### 2. 投资策略分享类  
**标题公式**: "普通人如何用[金额]开始[投资类型]？"
**结构**: 现状分析 → 策略介绍 → 风险提示 → 具体步骤

### 3. 市场分析解读类
**标题公式**: "[事件]对你的钱袋子有什么影响？"
**结构**: 事件概述 → 影响分析 → 应对策略 → 长期展望

### 4. 工具推荐类
**标题公式**: "这[数字]个免费金融工具，帮你省下[金额]"
**结构**: 痛点刺激 → 工具展示 → 使用教程 → 效果对比

## 爆款元素
- **具体数字**: "年化收益8.5%"、"节省2300元"
- **时间紧迫**: "现在不看就晚了"、"月底前必须知道"
- **权威背书**: 引用官方数据、知名机构观点
- **实用价值**: 提供可立即执行的具体建议
- **风险提示**: 合规的风险披露，建立信任

## 合规要求
- 必须包含"投资有风险，入市需谨慎"
- 不承诺具体收益
- 不推荐具体股票代码
- 区分事实陈述与个人观点
